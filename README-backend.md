# Booking backend (local)

## Requirements
- Node.js 18+ recommended

## Install
From this folder (`Larry website`):

```bash
npm install
```

## Run
```bash
npm start
```

Backend runs at:
- `http://localhost:3000/api/health`
- `POST http://localhost:3000/api/bookings`

## Where bookings are stored
- `./data/bookings.json`
